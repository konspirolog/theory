# Организации
Организации с помощью которых управляют народными массами.

- **Council on Foreign Relations**, CFR (Совет по международным отношениям) - частная американская организация в сфере международных связей. Учрежена в 1921м году при активном участии банкира Варбурга и советника президента Вудро Вильсона, полковника Хауса.
- **Bilderberg meeting** (Бильдербергский клуб) - создан в 1954г, для неформальных встреч представителей мировой финансовой, политической и интеллектуальной элит разных стран. 
- **Римский клуб** (Club of Rome) - создан в 1968м году. Международная интеллектуальная организация открытого типа. Актив Дэвида Рокфеллера.
- **Trilateral Commission** (Трёхсторонняя комиссия) - создан в 1973м году. Частная международная организация, состоящая из представителей Северной Америки, Западной Европы и Азии (Япония, Южная Корея) - обсуждение и поиск решений мировых проблем. Дополняет CFR и Bilderberg meeting. Создал Дэвид Рокфеллер.
